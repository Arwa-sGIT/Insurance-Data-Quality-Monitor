# Insurance Data Quality Monitor

## Project Status: In Progress (Publishing June 2026)

### Business Context
Simulating multi-entity data quality challenges in insurance operations. 
The dashboard tracks data completeness, accuracy, and consistency across 
distributed entities — mirroring enterprise-scale data governance.

### Objectives
- Measure data quality metrics at entity level
- Identify anomalies and data integrity issues
- Present findings in executive-friendly format

### Tools
- Python (Pandas, data generation)
- Power BI (Dashboard & Visualization)
- SQL (future)

### Key Metrics
- Data Completeness %
- Accuracy Score
- Duplicate Rate
- Entity-Level Comparison
